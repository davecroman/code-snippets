
## Create a local branch

```bash
git branch crazy-experiment
```

## Checkout a local branch

```bash
git checkout <branch-name>

```

## Checkout a remote branch

*Works only on Git >= 1.6.6*
```bash
git fetch
git checkout <branch name>
```

## List all branches

```bash
git branch
```

## Deleting a local branch

```bash
git branch -d the_local_branch
```
